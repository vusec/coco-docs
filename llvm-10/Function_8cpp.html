◆ getMangledTypeStr()

static std::string getMangledTypeStr ( Type *  Ty)
static

Returns a stable mangling for the type specified for use in the name mangling scheme used by 'any' types in intrinsic signatures.

The mangling of named types is simply their name. Manglings for unnamed types consist of a prefix ('p' for pointers, 'a' for arrays, 'f_' for functions) combined with the mangling of their component types. A vararg function type will have a suffix of 'vararg'. Since function types can contain other function types, we close a function type mangling with suffix 'f' which can't be confused with it's prefix. This ensures we don't have collisions between two unrelated function types. Otherwise, you might parse ffXX as f(fXX) or f(fX)X. (X is a placeholder for any other type.)
