Detailed Description

Helper for mapping JSON objects onto protocol structs.

Example:

bool fromJSON(const Value &E, MyStruct &R) {
ObjectMapper O(E);
if (!O || !O.map("mandatory_field", R.MandatoryField))
return false;
O.map("optional_field", R.OptionalField);
return true;
}
