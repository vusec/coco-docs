#define OMP_DIRECTIVE (   Enum,

)    Enum,

Make the enum values available in the llvm::omp namespace.

This allows us to write something like OMPD_parallel if we have a using namespace omp. At the same time we do not loose the strong type guarantees of the enum class, that is we cannot pass an unsigned as Directive without an explicit cast.
