class FDRTraceWriter

Declaration

class FDRTraceWriter : public RecordVisitor { /* full declaration omitted */ };

Description

The FDRTraceWriter allows us to hand-craft an XRay Flight Data Recorder (FDR) mode log file. This is used primarily for testing, generating sequences of FDR records that can be read/processed. It can also be used to generate various kinds of execution traces without using the XRay runtime. Note that this writer does not do any validation, but uses the types of records defined in the FDRRecords.h file.
